테스트 순서

-------------------------------------

회원가입

로그인

메인
- 평점 확인(평점 계산 제대로 됐는지 => ReviewBoard에서 해당하는 show의 평점 계산 제대로 됐는지)
- 클릭하면 다른 페이지로 제대로 이동되는지

공연 상세
- 날짜 선택 제대로 되는지
- 예매하기 눌렀을때 해당하는 예매번호로 가는지 확인하기
- 공연 후기 좋아요 눌리는지

예매
- 한 고객이 여러 브라우저로 해도 제대로 되는지
- 여러 고객이 여러 브라우저로 해도 제대로 되는지
- dB에 잘 저장되는지
- 선택된 좌석이 풀리는 일이 없는지

마이페이지
- 닉네임/비밀번호/프로필 이미지 변경 잘 되는지
- 예매 후기 평점 제대로 입력되는지,(메인에 반영되는지 확인)
- 예매 후기 수정 삭제 동작하는지, 좋아요 수 잘 불러오는지
- 내가 쓴 글, 내가 좋아요한 글로 잘 이동하는지

자유게시판
- 목록 제대로 출력되는지, 페이지네이션, 정렬 잘되는지
- 게시글 등록 제대로 되는지
- 좋아요/조회수/신고 반영 잘되는지
- 댓글, 대댓글 제대로 입력되는지

채팅
- 제대로 되는지

관리자페이지
- 공연 등록 수정 제대로 되는지
- 

로그아웃 되는지
